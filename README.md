# PS5 Sound Theme for KDE Plasma 6

### Prerequisites
1. git
2. VLC Media Player/FFMPEG
3. Linux distribution that supports KDE Plasma 6

### Installation
1. Use `git clone https://github.com/DAK404/PS5-Plasma-Soundtheme.git` to clone the repository to your disk
2. Change the directory to `PS5-Plasma-Soundtheme` by using `cd PS5-Plasma-Soundtheme`
3. [ OPTIONAL ] Make any necessary modifications to the sound files
4. Copy the directory based on the type of installation you want to perform:
    1. System-wide installation: `cp -r PS5-Plasma-Soundtheme /usr/share/sounds/`
    2. User level installation: `cp -r ~/.local/share/sounds/PS5-Plasma-Soundtheme`
5. Go to Plasma settings -> Colors and Themes -> System Sounds -> Change the theme to PS5
6. [ OPTIONAL ] Go to settings -> notifications -> System notifications -> customize as necessary
